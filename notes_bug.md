# ملاحظات خطأ الإجمالي

## المشكلة:
- إجمالي الشهر يعرض 25 ر.س (الكاش فقط)
- يجب أن يكون 25 + 850 = 875 ر.س (الكاش + الشبكة)

## البيانات المعروضة:
- إجمالي النقدي: 25
- إجمالي الشبكة: 850
- إجمالي الرصيد: 850
- إجمالي الشهر: 25 ر.س ← هذا خطأ!

## السبب:
في routers.ts سطر 1428:
```ts
const calculatedTotal = cashAmount; // الخطأ هنا - يحسب الكاش فقط
```

يجب أن يكون:
```ts
const calculatedTotal = cashAmount + networkAmount; // الكاش + الشبكة
```
